# graph-olap-sdk

Python SDK for Graph OLAP platform - client library for API interactions

## Part of Graph OLAP Platform

This repository contains the graph-olap-sdk component of the Graph OLAP platform.

## Quick Start

```bash
# Install dependencies
make install

# Run tests
make test

# Run linter
make lint
```

## Development

See `make help` for all available commands.
