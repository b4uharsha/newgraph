# graph-olap-schemas

Shared Pydantic schemas for Graph OLAP platform

## Part of Graph OLAP Platform

This repository contains the graph-olap-schemas component of the Graph OLAP platform.

## Quick Start

```bash
# Install dependencies
make install

# Run tests
make test

# Run linter
make lint
```
