# documentation

Starlight documentation site for Graph OLAP platform (replaces MkDocs per ADR-141)

## Part of Graph OLAP Platform

This repository contains the documentation component of the Graph OLAP platform.

## Quick Start

```bash
# Install dependencies
make install

# Run tests
make test

# Run linter
make lint

# Build Docker image
make build

# Deploy to local cluster
make deploy TARGET=local

# Check status
make status TARGET=local
```

See `make help` for all available commands.
