## Platform overview

The Graph OLAP Platform provides graph-based analytics over OLAP data sources.
It federates queries across Starburst (Trino), materialises results into
FalkorDB and RyuGraph for graph traversal, and exposes a Python SDK for
analysts working in JupyterHub. The platform runs on Kubernetes (GKE) with
Helm-managed deployments and ArgoCD-driven continuous delivery.
